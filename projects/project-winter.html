Project Winter

Technical Director • Multiplayer Social Deception Survival Game

Overview

Project Winter is an 8‑player social deception survival game where communication, teamwork, and betrayal shape every match. Players must gather resources, repair structures, and survive the elements—while traitors work from within to sabotage the group. The game blends cooperative survival with hidden‑role mechanics, creating unpredictable and highly replayable sessions.

My Role

  • Led engineering across gameplay, backend, and live operations for a cross‑platform multiplayer title.
  • Architected scalable networking systems supporting over one million players across PC and console.
  • Oversaw CI/CD pipelines, build automation, and deployment workflows for rapid iteration.
  • Optimized performance across Unity, console hardware, and cross‑platform networking conditions.
  • Collaborated with design, production, and QA to deliver features under tight live‑service timelines.

Technical Deep Dive

I designed the authoritative server model using a hybrid relay + dedicated server architecture to support low‑latency cross‑platform play. The system included:

  • Custom matchmaking with role weighting, social grouping, and region‑based routing.
  • Predictive movement and rollback compensation to handle unreliable network conditions.
  • Telemetry‑driven balancing tools enabling live tuning of roles, sabotage mechanics, and progression.
  • Automated crash reporting, health dashboards, and alerting for live‑service stability.
